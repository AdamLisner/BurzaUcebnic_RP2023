# Burza Učebnic

- Ročníkový projekt pro čtvrtý ročník mého pobytu v ústavu jménem Gymnázium Arabská

Tato aplikace je určena pro digitalizaci prodeje učebnic mezi studenty. Umožňuje je nakupovat a inzerovat.

## Použité technologie
- Node.js
- JavaScript
- Next.js
- React
- PostgreSQL
- Prisma
- Firebase
- VSCode
- GitHub, Git
